# Getting Started with This Template

Welcome to the enterprise web template! This guide will help you understand how to customize this template for your business needs.

## Quick Start

1. **Clone the repository** and install dependencies with `pnpm install`
2. **Replace placeholders** - Search for `[PLACEHOLDER_NAME]` format and replace with your actual values
3. **Update content** - Add your products, blog posts, and pages in the `content/` directory
4. **Customize styling** - Modify Tailwind configuration and component styles

## Project Structure

```
src/
├── app/           # Next.js App Router pages
├── components/    # Reusable UI components
├── lib/           # Utility functions and services
└── config/        # Configuration files
content/
├── posts/         # Blog posts (MDX)
├── products/      # Product pages (MDX)
└── pages/         # Static pages (MDX)
```

## Key Features

- **Internationalization** - Built-in support for English and Chinese
- **SEO Optimized** - Automatic meta tags, sitemaps, and structured data
- **Performance** - Optimized for Core Web Vitals
- **Accessibility** - WCAG 2.1 compliant components

## Next Steps

1. Review the configuration files in `src/config/`
2. Update translation files in `messages/`
3. Add your products and blog content
4. Deploy to your preferred hosting platform

Happy building!
