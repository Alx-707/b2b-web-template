# 模板使用入门指南

欢迎使用企业网站模板！本指南将帮助您了解如何为业务需求定制此模板。

## 快速开始

1. **克隆仓库**并使用 `pnpm install` 安装依赖
2. **替换占位符** - 搜索 `[PLACEHOLDER_NAME]` 格式并替换为实际值
3. **更新内容** - 在 `content/` 目录中添加您的产品、博客文章和页面
4. **自定义样式** - 修改 Tailwind 配置和组件样式

## 项目结构

```
src/
├── app/           # Next.js App Router 页面
├── components/    # 可复用的 UI 组件
├── lib/           # 工具函数和服务
└── config/        # 配置文件
content/
├── posts/         # 博客文章 (MDX)
├── products/      # 产品页面 (MDX)
└── pages/         # 静态页面 (MDX)
```

## 主要特性

- **国际化** - 内置英语和中文支持
- **SEO 优化** - 自动生成元标签、站点地图和结构化数据
- **性能优化** - 针对 Core Web Vitals 进行优化
- **无障碍访问** - 符合 WCAG 2.1 标准的组件

## 下一步

1. 查看 `src/config/` 中的配置文件
2. 更新 `messages/` 中的翻译文件
3. 添加您的产品和博客内容
4. 部署到您选择的托管平台

祝您开发顺利！
